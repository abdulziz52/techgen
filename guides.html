Step-by-Step: Your First Website

Step 1: Set up your environment

Download VS Code (Visual Studio Code). It's the best free code editor used by pros.

Step 2: Create your files

Create a folder on your desktop. Inside, create a file named index.html.

Step 3: Write some HTML

Type <h1>Hello World</h1> in your file and save it.

Step 4: See it live

Double click your index.html file. It will open in your browser. Congratulations, you just wrote code!
